# RIALÉ RECORDS — Prototype statique (avec tes liens)

Ce prototype intègre les liens Facebook et YouTube que tu as fournis, et utilise +226 55275717 comme contact/WhatsApp.

Fichiers :
- index.html : page principale (contact mis à jour)
- styles.css : styles (inchangés)
- artists.json : artistes remplis avec les URLs que tu as données (bios placeholders)

Modifier les artistes :
- Ouvre artists.json et remplace "name", "bio", "image" et "track" par les vraies informations.
- Pour ajouter une piste audio, héberge l'audio sur SoundCloud / un CDN public et mets l'URL dans "track".

Déployer gratuitement :
- GitHub Pages : créer un dépôt public, uploader ces fichiers, Settings > Pages > branch main / root.
- Netlify : lier au dépôt GitHub et déployer.

Besoin d'aide ?
- Donne-moi les vrais noms et courtes bios (1-2 phrases) pour chaque lien et je mets à jour le fichier.
- Si tu veux, envoie des images (liens directs) et je les intègre.
